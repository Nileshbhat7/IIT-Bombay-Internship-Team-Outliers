# 📌 Project Overview

This repository contains the work completed during the **IIT Bombay Internship Program** by **Team Outliers**.

The project focuses on analyzing a multimodal dataset collected during mental rotation tasks and applying **unsupervised machine learning techniques** to identify hidden participant groups based on physiological and behavioral characteristics.

---

# 🎯 Problem Statement

**Unsupervised Clustering of Participants**

The objective of this project is to discover latent participant groups using multimodal physiological and behavioral data through clustering techniques.

---

# 📊 Dataset

The dataset contains multiple data modalities, including:

- EEG Signals
- Galvanic Skin Response (GSR)
- Accelerometer Data
- Gyroscope Data
- Facial Expressions
- Emotion Logs
- Task Performance Metrics
- Participant Metadata

---

# 🧠 Machine Learning Workflow

- Data Preprocessing

- Exploratory Data Analysis (EDA)
